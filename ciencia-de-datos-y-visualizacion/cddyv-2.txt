```python
from textblob import TextBlob

def analizar_sentimiento(texto):
    """
    Analiza el sentimiento de una cadena de texto y la clasifica
    como Positivo, Negativo o Neutro.

    Args:
        texto (str): La frase o texto a analizar.

    Returns:
        str: El resultado de la clasificaciÃ³n del sentimiento.
    """
    # 1. Crear un objeto TextBlob
    blob = TextBlob(texto)

    # Nota: TextBlob realiza el anÃ¡lisis de sentimiento en inglÃ©s de forma nativa.
    # Para trabajar con espaÃ±ol, primero debemos traducirlo.
    try:
        # 2. Intentar la traducciÃ³n a inglÃ©s (si el idioma original no es inglÃ©s)
        # TextBlob intenta detectar el idioma. Si no es 'en', lo traduce.
        if blob.detect_language() != 'en':
             blob = blob.translate(to='en')
        
    except Exception as e:
        # En caso de error de detecciÃ³n/traducciÃ³n, usamos el texto original
        # y emitimos una advertencia si el idioma no es inglÃ©s.
        print(f"Advertencia: Error en la traducciÃ³n ({e}). Analizando el texto original.")
        # El anÃ¡lisis funcionarÃ¡ para frases simples en espaÃ±ol, pero es menos preciso.

    # 3. Obtener la Polaridad del sentimiento
    # La polaridad es un valor flotante en el rango [-1.0, 1.0].
    # -1.0 es completamente negativo, +1.0 es completamente positivo.
    polaridad = blob.sentiment.polarity
    
    # 4. Clasificar el sentimiento basado en la polaridad
    if polaridad > 0.1:
        sentimiento = "Positivo"
    elif polaridad < -0.1:
        sentimiento = "Negativo"
    else:
        sentimiento = "Neutro"
        
    # 5. Devolver el resultado
    return f"Texto: '{texto}'\nPolaridad: {polaridad:.2f}\nSentimiento: **{sentimiento}**"
```

#### 2\. Funcionamiento de `TextBlob`

  * **`TextBlob(texto)`**: Crea un objeto `TextBlob` que envuelve el texto, dÃ¡ndole acceso a varias propiedades de NLP.
  * **DetecciÃ³n y TraducciÃ³n (Para EspaÃ±ol)**: Por defecto, `TextBlob` entrena sus modelos de sentimiento en inglÃ©s. Para analizar texto en espaÃ±ol, la estrategia es:
    1.  Detectar el idioma original (`blob.detect_language()`).
    2.  Si no es inglÃ©s (`'en'`), traducirlo a inglÃ©s (`blob.translate(to='en')`).
  * **`blob.sentiment.polarity`**: Es el valor clave. Representa la fuerza emocional del texto:
      * **$1.0$**: MÃ¡ximo sentimiento positivo.
      * **$-1.0$**: MÃ¡ximo sentimiento negativo.
      * **$0.0$**: Sentimiento neutro.

#### 3\. ClasificaciÃ³n

En el cÃ³digo, he usado umbrales de $\pm 0.1$ para clasificar el sentimiento, ya que una polaridad de $0.00$ a $\pm 0.10$ se considera generalmente neutral en la prÃ¡ctica:

  * **Positivo:** Si $Polaridad > 0.1$
  * **Negativo:** Si $Polaridad < -0.1$
  * **Neutro:** Si $-0.1 \le Polaridad \le 0.1$
