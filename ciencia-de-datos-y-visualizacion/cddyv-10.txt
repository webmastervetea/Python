# La b√∫squeda binaria es un algoritmo eficiente para encontrar un elemento en una lista **ordenada**.

# \# Su eficiencia se debe a que divide repetidamente a la mitad la porci√≥n de la lista donde podr√≠a estar el elemento.

# \# La complejidad de tiempo de la b√∫squeda binaria es $O(\log n)$, lo cual es mucho m√°s r√°pido que la b√∫squeda lineal $O(n)$ para listas grandes.

```python
def busqueda_binaria(lista_ordenada, objetivo):
    """
    Implementa el algoritmo de b√∫squeda binaria.

    Asume que la lista de entrada est√° ordenada de forma ascendente.

    :param lista_ordenada: La lista ordenada donde se realizar√° la b√∫squeda.
    :param objetivo: El valor que se est√° buscando.
    :return: El √≠ndice del objetivo en la lista si se encuentra, o -1 si no se encuentra.
    """
    # 1. Inicializar los punteros de los l√≠mites del rango de b√∫squeda
    # # `izquierda` apunta al inicio de la lista (√≠ndice 0)
    izquierda = 0
    # # `derecha` apunta al final de la lista (√∫ltimo √≠ndice)
    derecha = len(lista_ordenada) - 1

    # 2. Ciclo principal de b√∫squeda
    # # El ciclo contin√∫a mientras el rango de b√∫squeda sea v√°lido (izquierda <= derecha)
    while izquierda <= derecha:
        # # Calcular el √≠ndice del medio.
        # # Se usa la f√≥rmula `(izquierda + derecha) // 2` para asegurar que el resultado sea un entero (divisi√≥n entera).
        # # Tambi√©n es una forma segura de evitar desbordamientos de enteros en otros lenguajes.
        medio = (izquierda + derecha) // 2

        # # Obtener el valor en el punto medio
        valor_medio = lista_ordenada[medio]

        # 3. Comparaci√≥n con el objetivo
        if valor_medio == objetivo:
            # # ¬°Elemento encontrado! Devolver el √≠ndice.
            return medio
        elif valor_medio < objetivo:
            # # El objetivo es MAYOR que el valor del medio.
            # # Esto significa que el objetivo solo puede estar en la mitad DERECHA de la lista.
            # # Mover el puntero izquierdo a `medio + 1` para buscar en la mitad superior.
            izquierda = medio + 1
        else: # valor_medio > objetivo
            # # El objetivo es MENOR que el valor del medio.
            # # El objetivo solo puede estar en la mitad IZQUIERDA de la lista.
            # # Mover el puntero derecho a `medio - 1` para buscar en la mitad inferior.
            derecha = medio - 1

    # 4. Finalizaci√≥n del ciclo
    # # Si el ciclo termina sin encontrar el elemento, significa que el objetivo no est√° en la lista.
    return -1

# --- Bloque Principal de Ejecuci√≥n ---

# # Ejemplo de lista de entrada (¬°DEBE ESTAR ORDENADA!)
MI_LISTA = [2, 5, 8, 12, 16, 23, 38, 56, 72, 91]

# # Definir el elemento que queremos buscar
OBJETIVO_1 = 23 # Un elemento que s√≠ est√° en la lista
OBJETIVO_2 = 42 # Un elemento que NO est√° en la lista

print("# --- ALGORITMO DE B√öSQUEDA BINARIA ---")
print(f"# Lista: {MI_LISTA}")

# # B√∫squeda 1: Elemento presente
indice_1 = busqueda_binaria(MI_LISTA, OBJETIVO_1)

if indice_1 != -1:
    print(f"\n# B√∫squeda de {OBJETIVO_1}: ¬°Encontrado!")
    print(f"# El elemento est√° en el √≠ndice: {indice_1}")
else:
    print(f"\n# B√∫squeda de {OBJETIVO_1}: No encontrado.")

# # B√∫squeda 2: Elemento ausente
indice_2 = busqueda_binaria(MI_LISTA, OBJETIVO_2)

if indice_2 != -1:
    print(f"\n# B√∫squeda de {OBJETIVO_2}: ¬°Encontrado!")
    print(f"# El elemento est√° en el √≠ndice: {indice_2}")
else:
    print(f"\n# B√∫squeda de {OBJETIVO_2}: No encontrado.")

print("# ----------------------------------------")
```

## üìù Concepto Clave de la B√∫squeda Binaria

# La **B√∫squeda Binaria** se basa en tres pasos fundamentales que se repiten en cada iteraci√≥n:

# 1\. **Punto Medio:** Se encuentra el √≠ndice del elemento central del subarreglo actual.

# 2\. **Comparaci√≥n:** Se compara el elemento central con el objetivo.

# 3\. **Reducci√≥n:** Si no son iguales, se descarta el 50% de la lista donde el objetivo no puede estar, moviendo el puntero `izquierda` o `derecha`.
