# Este script requiere que la librer√≠a **keyboard** est√© instalada y que se ejecute con **privilegios suficientes** (puede requerir `sudo` en Linux o privilegios de administrador en Windows, dependiendo de la configuraci√≥n).

# \# Puedes instalarla con `pip install keyboard`.

```python
import keyboard
import os
import subprocess
import time

# --- CONFIGURACI√ìN ---

# # Diccionario de aplicaciones y comandos a ejecutar.
# # # CLAVE: Atajo de teclado a presionar (Ej: 'ctrl+alt+n', 'win+p').
# # # VALOR: El comando del sistema operativo para abrir la aplicaci√≥n o archivo.
# # # NOTA IMPORTANTE: Los comandos deben ser v√°lidos para tu sistema operativo (Windows/Linux/macOS).
ATAJOS_DE_LANZAMIENTO = {
    # Ejemplo 1: Abrir el Bloc de Notas (Windows) o Gedit (Linux)
    'ctrl+alt+n': 'notepad.exe', # Para Windows
    # 'ctrl+alt+n': 'gedit', # Para Linux
    
    # Ejemplo 2: Abrir el Navegador Web
    'ctrl+alt+w': 'start chrome "https://www.google.com"', # Para Windows (abre Chrome)
    # 'ctrl+alt+w': 'xdg-open "https://www.google.com"', # Para Linux (abre el navegador por defecto)
    
    # Ejemplo 3: Abrir la Calculadora
    'ctrl+alt+c': 'calc.exe', # Para Windows
    # 'ctrl+alt+c': 'gnome-calculator', # Para Linux (cambiar seg√∫n tu DE)
    
    # Ejemplo 4: Abrir una carpeta espec√≠fica (reemplaza la ruta)
    'ctrl+alt+f': 'explorer C:\\Users\\Public\\Documents', # Para Windows
    # 'ctrl+alt+f': 'xdg-open /home/usuario/Documentos', # Para Linux (reemplaza la ruta)
}

def ejecutar_comando(comando):
    """
    Ejecuta un comando del sistema operativo de forma as√≠ncrona.

    :param comando: La cadena de comando a ejecutar.
    """
    try:
        # # `subprocess.Popen` inicia la aplicaci√≥n sin esperar a que termine,
        # # # lo cual es ideal para un lanzador de aplicaciones.
        subprocess.Popen(comando, shell=True)
        print(f"# ‚úÖ Comando ejecutado: {comando}")
    except Exception as e:
        print(f"# ‚ùå ERROR al ejecutar '{comando}': {e}")
        # # Si falla, podr√≠a ser por un comando incorrecto o falta de privilegios.
        messagebox.showerror("Error de Ejecuci√≥n", f"No se pudo ejecutar: {comando}\nError: {e}")

def configurar_lanzador():
    """
    Configura y registra todos los atajos de teclado definidos.
    """
    print("# --- LANZADOR DE APLICACIONES R√ÅPIDO ---")
    print("# El lanzador est√° escuchando los siguientes atajos:")
    
    # # Iterar sobre el diccionario de atajos y comandos
    for atajo, comando in ATAJOS_DE_LANZAMIENTO.items():
        # # `keyboard.add_hotkey(atajo, funci√≥n)` registra el atajo.
        # # # Cuando el atajo es presionado, se llama a la funci√≥n lambda que ejecuta el comando.
        keyboard.add_hotkey(atajo, lambda cmd=comando: ejecutar_comando(cmd))
        print(f"# [{atajo}] -> Comando: {comando}")
    
    print("\n# üü¢ Lanzador activo. Presione Ctrl+C en la terminal para detener.")
    
    # 3. Mantener el programa corriendo
    try:
        # # `keyboard.wait()` bloquea el script hasta que se presione una tecla especial 
        # # # o hasta que el programa sea interrumpido manualmente (Ctrl+C).
        keyboard.wait('esc') # Podr√≠as esperar 'esc' o simplemente dejar un bucle.
        # # Una alternativa simple es un bucle infinito que se detiene con Ctrl+C
        # while True:
        #     time.sleep(1) 
            
    except KeyboardInterrupt:
        print("\n# üõë Lanzador detenido por el usuario.")
    except Exception as e:
        print(f"\n# üõë Error inesperado: {e}")
        
    print("# --- FIN DEL PROGRAMA ---")


# --- Bloque Principal de Ejecuci√≥n ---
if __name__ == "__main__":
    # # Importar messagebox solo si es necesario (para evitar dependencia de Tkinter si no se usa)
    try:
        from tkinter import messagebox
    except ImportError:
        # # Si Tkinter no est√° disponible, creamos una funci√≥n dummy para evitar errores
        print("# Advertencia: Tkinter no encontrado. Las alertas de error no se mostrar√°n.")
        def messagebox(*args, **kwargs): pass
        
    configurar_lanzador()
```

## üìù Instrucciones y Advertencias

1.  **Instala la librer√≠a `keyboard`:**
    ```bash
    pip install keyboard
    ```
2.  **Ajusta los comandos:** **Debes modificar** el diccionario `ATAJOS_DE_LANZAMIENTO` para que los comandos (`notepad.exe`, `start chrome`, `gedit`, `xdg-open`, etc.) coincidan con las aplicaciones instaladas y las sintaxis de comandos de tu **Sistema Operativo** (Windows, Linux o macOS).
3.  **Guarda y Ejecuta:** Guarda el c√≥digo en un archivo (`lanzador.py`) y ejec√∫talo desde tu terminal:
    ```bash
    python lanzador.py
    ```
      * **Advertencia de Permisos:** En algunos sistemas (especialmente al usar combinaciones como `Win` o `Ctrl+Alt`), puede que necesites ejecutar el script con permisos de administrador o `sudo` para que pueda capturar los atajos de teclado globales.
4.  **Usa los Atajos:** Mientras el script est√© corriendo en la terminal, presiona las combinaciones de teclas que definiste (ej. `Ctrl + Alt + N`).
5.  **Det√©n el Lanzador:** Vuelve a la terminal y presiona **`Ctrl + C`** para detener el programa.
